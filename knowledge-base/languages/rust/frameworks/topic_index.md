# Rust Frameworks Index

Curated notes on Rust frameworks used to build applications, bots, and services.

- [Teloxide (Rust Telegram Bot Framework)](teloxide.md) — Async-first toolkit
  for wiring Telegram Bot API bots with typed dispatching, long polling, or
  webhooks.
- [RMCP (Rust Remote Management Control Protocol crate)](rmcp.md) — Packet
  models and helpers for crafting ASF/IPMI traffic over RMCP in Rust services.
- [rudof — Rust RDF validation toolkit](rudof.md) — CLI/library plus Python
  bindings for ShEx, SHACL, and DCTAP validation and conversion flows.
